# Armshift

  

a simple sim game made in 10 days as an entry for a Gamedev.js game jam in 2026.

  

## Description

  

The game is about a robotic arm that works a shift in a factory, your job is to take the cargo boxes, put them in the truck, and deliver them.
some bad tiny robots will come and try to destroy your cargo, you can eliminate them with a laser weapon installed in the arm.

  

## Getting Started

  

### Dependencies

  

* Only godot is required (Godot 4.6 stable is used). 
  

### Executing program

  

* The repo contains both the source code and a pre-built web game placed in "build"


## Help

  

Some features will not work such as Global leaderboards as its' dependent on wavedash, to utilize it navigate to [Wavedash docs](https://docs.wavedash.com/sdk/) docs for more on how to integrate.
